Class Entity

Represents any object of the world.

Components may be attached to it to add behaviors (like a position, a renderer or a physics body).

Tables

self.components All added components are accessible through this table.

Fields

self.uid The unique generated identifier of the entity.
self.isVisible Indicates whether the entity is visible.
self.isDestroyed Indicates the entity is destroyed and should be removed from the world.

Methods

Entity:clone () Clones the entity.
Entity:serialize () Extracts the entity state as a serializable table.
Entity:deserialize (data) Restores the state of an entity from a state table.
Entity:has (component) Indicates whether this entity has a component of the given class.
Entity:get (component) Gets the entity with given component type.
Entity:add (component) Adds the component to the entity.
Entity:remove (component) Removes the component to the entity.


Tables

self.components
All added components are accessible through this table. Note that their keys are the name of their class with a lowercase first letter (i.e. "Position" > "components.position")

Fields

self.uid
The unique generated identifier of the entity.
self.isVisible
Indicates whether the entity is visible.
self.isDestroyed
Indicates the entity is destroyed and should be removed from the world.

Methods

Entity:clone ()
Clones the entity. It creates an entity of the same class and clone each of its components.

Returns:

    The newly cloned entity.
Entity:serialize ()
Extracts the entity state as a serializable table. All information for this entity to be restored is present in this table : uid, component stores, ...

Returns:

    A state table.
Entity:deserialize (data)
Restores the state of an entity from a state table.

Parameters:

  • data
Entity:has (component)
Indicates whether this entity has a component of the given class.

Parameters:

  • component A component class

Returns:

    false if the entity hasn't any component of the given component type.
Entity:get (component)
Gets the entity with given component type.

Parameters:

  • component A component class

Returns:

    The component if owned, else nil
Entity:add (component)
Adds the component to the entity. Note: an entity can only have one component of each type.

Parameters:

  • component The component instance.
Entity:remove (component)
Removes the component to the entity.

Parameters:

  • component The component instance, or type.
